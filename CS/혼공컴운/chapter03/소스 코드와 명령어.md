# 고급 언어와 저급 언어

* 프로그램을 만들 때 사용하는 프로그래밍 언어는 컴퓨터가 이해하는 언어가 아닌 사람이 이해하고 작성하기 쉽게 만들어진 언어이다.

## 고급언어 
사람이 쉽게 사용할 수 있는 대부분의 프로그래밍 언어를 의미
ex) JAVA, Python, C# 등

## 저급언어
컴퓨터가 직접 이해하고 실행할 수 있는 언어
ex) 기계어, 어셈블리어

# 컴파일 언어와 인터프리터 언어

* 고급언어로 작성된 소스코드는 결국 저급언어로 변환되어 실행된다. 
고급언어가 저급언어로 변환되는 방법에는 크게 컴파일, 인터프리트 두 가지 방식이 존재한다.

## 컴파일 언어 
컴파일러에 의해 소스 코드 전체가 저급 언어로 변환되어 실행되는 고급 언어이다.
대표적인 컴파일 언어로는 C언어가 존재한다.
소스코드가 저급 언어로 변환되는 과정을 컴파일이라고 하며, 이 과정을 도와주는 도구를 컴파일러라고 한다.
컴파일러는 개발자가 작성한 소스 코드 전체를 확인하여 소스 코드에에 문법적인 오류는 없는지, 실행 가능한 코드인지, 실행하는데 불필요한 코드는 없는지 등을 따지며 이때 하나라도 오류를 발견하면 소스 코드는 컴파일에 실패한다.
