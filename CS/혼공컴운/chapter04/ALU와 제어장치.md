# ALU
* ALU는 CPU안의 계산을 수행하는 부품이다.
* ALU가 계산을 하기 위해서는 피연산자와, 수행할 연산이 필요하다.
* ALU는 레지스터와 제어장치로부터 받아들인 피연산자와 제어 신호로 산술 연산, 논리 연산 등 다양한 연산을 수행한다.
* CPU가 메모리에 접근하는 속도는 레지스터에 접근하는 속도보다 훨씬 느리다. 그래서, ALU가 수행한 연산의 결과값은 일시적으로 메모리가 아닌 레지스터에 저장된다.

# 제어장치
* 제어 신호를 내보내고, 명령어를 해석하는 부품이다.
* 제어 신호는 컴퓨터 부품들을 관리하고 작동시키기 위한 일종의 전기 신호이다.

## 제어장치가 받아들이는 정보
1. 제어장치는 '클럭' 신호를 받아들인다.
- 클럭이란 컴퓨터의 모든 부품을 움직일 수 있는 시간 단위로, 컴퓨터의 모든 부품은 클럭 신호에 맞춰 동작한다.

2. 제어 장치는 '해석하야 할 명령어'를 받아들인다.
- CPU가 해석해야 할 명령어는 명령어 레지스터에 저장된다. 제어장치는 이 명령어 레지스터로부터 해석해야 할 명령어를 받아들이고 해석한 후, 제어신호를 발생시켜 컴퓨터 부품들에 수행해야 할 내용을 알려준다.

3. 제어 장치는 플래그 레지스터 속 플래그 값을 받아들인다.
- 플래그는 ALU에 연산에 대한 추가정인 상태 정보로, 제어장치 또한 이 플래그 값을 받아들이고 참고하여 제어 신호를 발생시킨다.

4. 제어장치는 시스템 버스 중 제어 버스로 전달된 제어 신호를 받아들인다.
- 제어 신호를 CPU뿐만 아닌 입출력장치를 비롯한 CPU외부 장치도 발생시킬 수 있다.
- 제어장치는 제어 버스를 통해 외부로부터 전달되니 제어 신호를 받아들이기도 한다.

## 제어장치가 내보내는 정보
1. CPU 외부에 전달하는 제어 신호
- 제어 버스로 제어 신호를 메모리, 입출력장치에 전달한다.
- 제어 장치가 메모리에 저장된 값을 읽거나 메모리에 새로운 값을 쓰고 싶다면 메모리로 제어신호를 내보낸다.
- 제어장치가 입출력장치의 값을 읽거나 입출력장치에 새로운 값을 쓰고 싶을 때는 입출력장치로 제어신호를 내보낸다.

2. CPU 내부에 전달하는 제어 신호
- 수행할 연산을 지시하기 위해 ALU에 전기 신호를 내보낸다.
- 레지스터 간 데이터를 이동시키거나 레지스터에 저장된 명령어를 해석하기 위해 레지스터에 전기 신호를 내보낸다.
