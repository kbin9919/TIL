# 문자 집합과 인코딩

## 문자 집합
컴퓨터가 인식하고 표현할 수 있는 문자의 모음을 **문자집합**이라고 한다.
컴퓨터는 문자집합에 속해있는 문자를 이해할 수 있고, 반대로 속해있지 않으면 이해할 수 없다.

## 문자 인코딩
문자 집합에 속해 있다고 해서, 컴퓨터가 그대로 이해할 수 있는 건 아니다.
컴퓨터가 문자를 이해하기 위해서는 문자를 0과 1로 변환해야한다. 이 변환 과정을 문자 인코딩이라고 한다.

## 문자 디코딩
인코딩의 반대 과정, 즉 0과 1로 이루어진 문자 코드를 사람이 이해할 수 있는 문자로 변환하는 과정을 의미한다.

# 아스키 코드
초창기 문자 집합 중 하나로, 영어 알파벳과 아라비아 숫자, 그리고 일부 특수 문자를 포함한다.
아스키 문자 집합에 속한 문자들은 각각 7비트로 포현되는데, 7비트로 표현될 수 있는 정보의 가짓수는 128개로 총 128개의 문자를 표현할 수 있다.

# EUC-KR
아스키 코드로는 표현 가능한 숫자가 제한되어서 한글을 표현할 수 없다.
**EUC-KR**은 KS X 1001, KS X 1003이라는 문자 집합을 기반으로 하는 대표적인 완성형 인코딩이다.
초성, 중성, 종성이 모두 결합된 한글 단어에 2바이트 크기의 코드를 부여한다.
총 2350개 정도의 한글 단어를 표현할 수 있으나, 문자 집합에 포함되지 않은 '뷁', '뷇'와 같은 글자는 표현할 수 없다.

# 유니코드
유니코드는 EUC-KR보다 훨씬 다양한 한글을 포함하며 대부분 나라의 문자, 특수문자, 화살표나 이모티콘까지도 코드로 표현할 수 있는 통일된 문자집합이다. 유니코드는 현대 문자를 표현할 때 가장 많이 사용되는 표준 문자 집합이다.

# UTF
유니코드를 인코딩 하는 방법으로, 크게 UTF-8, UTF-16, UTF-32등이 있다.
