# 트리(Tree)

* 트리는 **계층 구조를 표현하는 비선형 자료구조**이다.
* 하나의 루트(root) 노드를 중심으로 가지(branch)가 뻗어나가는 형태를 가지며, **부모–자식 관계**로 연결된 노드들의 집합이다.

---

# 트리의 특징

* 루트
- 루트(root)는 트리 구조에서 가장 최상단에 있는 시작 노드이다.
- 모든 노드의 “조상”이 되는 유일한 노드이며, 트리 전체가 이 루트를 중심으로 뻗어나간다.

* **루트 노드가 존재한다**
* 모든 노드는 **0개 이상의 자식 노드**를 가질 수 있다.
* 노드 간 연결은 **부모 → 자식 방향의 단방향**이다
* 사이클이 없다 (순환 구조 금지)
* 계층적 구조를 표현하는 데 최적화된 자료구조이다.

---

# 트리 구조 예시

```
        루트
       /    \
     A        B
    / \      / \
  C   D    E    F
```

* 파일 시스템 (폴더 구조)

* HTML DOM 구조

* DB 인덱스(B-Tree, B+Tree 등)
